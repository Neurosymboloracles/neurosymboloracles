# neurosymboloracles-
Art neuro-symbolique : fusion des créativité de réseaux de neurones (Pytorch) et de la précision des mathématiques pures (Sympy). Un modèle Pytorch génère une base visuelle ( ex. texture, forme). Sympy résout une équation liée. Les deux résultats fusionnent dans l'oeuvre finale, reproductible et vérifiable. Chaque artéfact change au 5e transfert.
